# 🔀 Git 命令生成器 (Git Command Generator)

> **中文描述生成命令 · 高频场景速查 · 本地风险检测 · 纯前端单文件**
>
> 基于纯原生前端打造的 Git 命令安全辅助工具。中文描述需求即可生成可执行命令，配合**本地规则引擎**做风险检测，AI 不参与任何安全判断——即使 AI 生成的命令有误，最终展示给你的风险等级永远是确定性的本地规则结果。

## ✨ 核心特性

* **三栏分工，各司其职**：
  * **📝 描述生成**：中文描述需求，AI 生成 1-3 个候选命令方案。
  * **🔍 场景速查**：五大高频分类的静态命令库，零 AI、不联网，支持中英文模糊搜索。
  * **🛡️ 风险检测**：粘贴任意命令，本地规则库实时检测高危模式，给出更安全的替代方案。
* **意图分流卡片**：当中文描述存在多种合理实现方式时（如"撤销commit"可能对应 `reset --soft` / `reset --hard` / `revert` 三种完全不同的语义），AI 会把每种方式作为独立卡片列出，并清楚说明各自的适用场景和后果差异，绝不强行给出单一答案制造选择陷阱。
* **AI 输出不被信任为安全判断依据**：风险等级和高危参数高亮**始终由本地规则引擎重新计算**，AI 只负责生成候选命令和场景说明，安全性判断完全锁定在已经过测试验证的本地规则。
* **状态假设开关**：把"工具无法感知你的真实仓库状态"这一技术局限，转化为两个可勾选的交互开关——
  * `[ ] 假设该分支已经推送到远程共享仓库`
  * `[ ] 假设当前本地工作区有未提交的临时代码`
  
  勾选后，本地规则引擎会动态将相关命令的风险等级升级（如 `push --force` 在"已推送远程"勾选后从黄色警告升级为黑红色"极危"），两个开关的状态在描述生成与风险检测两栏间保持同步。
* **Token 级高危参数高亮**：只对命令中的危险参数本身（如 `--force`、`--hard`）应用醒目样式，命令主干保持常规颜色，避免全行标红的视觉疲劳，复制前一眼看清"毒点"。
* **一键替换闭环**：检测到高危命令时，点击按钮直接用更安全的替代命令覆盖输入框，完成"检测→纠错→导出"的单流向闭环。
* **极简且安全**：场景速查与风险检测两个核心功能完全不依赖 API Key、不联网，纯本地计算；仅描述生成功能调用 AI。

## 📚 风险规则库覆盖

**⚠️ 高危命令检测：**

* `push --force` / `-f`：无条件覆盖远程历史，可能导致他人提交永久丢失。已推送远程时升级为极危。
* `reset --hard`：丢弃工作区和暂存区所有未提交修改，不可恢复。有未提交修改时升级为高风险。
* `clean -fd`：删除所有未跟踪文件和目录，不可恢复。有未提交修改时升级为高风险。
* `branch -D`：强制删除分支，即使有未合并提交也直接删除。
* `rebase`：改写提交历史。已推送远程时升级为极危。
* `commit --amend`：改写最近一次提交的 hash。已推送远程时升级为高风险。
* `checkout --force`：丢弃工作区中与目标分支冲突的未提交修改。

**✅ 安全替代方案：**

* `push --force` → `push --force-with-lease`（推送前检查远程是否有未知新提交）
* `reset --hard` → `reset --soft`（保留修改，只移动 HEAD 指针）
* `clean -fd` → `clean -fdn`（先预览会删除的文件，dry-run）
* `branch -D` → `branch -d`（有未合并提交时会拒绝删除并提示）

**🆕 现代命令版本兼容性标注：**

* `git switch` / `git restore`（Git 2.23+，checkout 的现代替代写法）
* `--force-with-lease`（Git 1.8.5+）

## 🧪 测试覆盖

* **风险检测引擎**：30 个单元测试，覆盖状态开关升级逻辑、`--force` vs `--force-with-lease` 区分、`rebase --abort/--continue` 排除、token 位置精确性等场景。
* **端到端测试（Playwright）**：16 项检查覆盖三栏核心交互、状态开关跨栏同步、一键替换闭环、模糊搜索、深色模式、API Key 管理，全程零 JS 报错。

## ⚠️ 已知限制

* 风险检测基于命令文本模式匹配，无法感知用户的真实仓库状态——这正是"状态假设开关"存在的原因。
* 规则库覆盖范围有限，未收录的命令模式不会触发任何风险提示。
* 不连接用户本地仓库，也无法连接（浏览器环境本身就做不到），纯命令生成/解释/检测。
